# live-kanban
live Kanban using Next Js


# Project Setup
NODE Version : v20.18.1
NVM Version  : v1.1.11

# For Start Project
npm i 
start project : npm run dev 

# Key Features
State Management:
    Redux Toolkit (RTK) for managing global state.
State Persistence:
    Redux Persist to persist authentication and state across page reloads.
